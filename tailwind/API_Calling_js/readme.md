This is a small weather app made using simple HTML and JavaScript.  
You can enter any city name and get the current weather details.

## What this app does
- You type a city name  
- It shows:
  - City, state, country  
  - Local time  
  - Temperature in Celsius  
  - Wind speed  

## Tech used
- HTML  
- JavaScript  
- Weather API from :contentReference[oaicite:0]{index=0}  

## How to use
1. Open `index.html` in your browser  
2. Enter a city name (like London, Toronto, Delhi)  
3. Click the **Search** button  
4. Weather details will appear on the screen
